# 翻译评测协调器

## 概述

协调器负责串联翻译评测系统的完整流程，管理迭代循环。

## 完整流程

```
┌─────────────────────────────────────────────────────────┐
│                    迭代流程                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐         │
│   │ 翻译     │───▶│ 评测     │───▶│ 优化     │         │
│   │ translate│    │ evaluate │    │ optimize │         │
│   └──────────┘    └──────────┘    └──────────┘         │
│        ▲                                 │              │
│        └─────────────────────────────────┘              │
│                    迭代循环                              │
└─────────────────────────────────────────────────────────┘
```

## 使用方式

### 单次运行

用户说："运行一轮评测"或"run one iteration"

执行步骤：
1. **翻译阶段** - 调用 trans-translate skill
2. **评测阶段** - 调用 trans-evaluate skill
3. **输出结果** - 报告当前准确率

### 完整迭代

用户说："开始迭代优化"或"run optimization loop"

执行步骤：
1. 运行翻译→评测→优化
2. 用新 Prompt 再次翻译→评测
3. 对比前后准确率
4. 决定是否继续迭代

### 停止条件

- 准确率 > 90%
- 连续 3 轮无提升
- 用户明确要求停止

## 协调器职责

### 1. 状态管理

维护 `~/.openclaw/workspace/translation-eval/state.json`：

```json
{
  "current_prompt_version": "v0.2",
  "current_accuracy": 0.78,
  "iteration_count": 2,
  "best_accuracy": 0.78,
  "best_version": "v0.2",
  "last_run": "2026-03-13T12:00:00Z"
}
```

### 2. 流程调度

```
开始 → 检查状态 → 翻译 → 评测 → 记录结果
                              ↓
                    准确率达标？ ──Yes→ 结束
                         ↓No
                      优化 Prompt
                         ↓
                    新一轮翻译...
```

### 3. 报告汇总

每次迭代后生成汇总报告：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  迭代 #2 完成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Prompt: v0.1 → v0.2
  准确率: 65% → 78% (+13%)

  主要改进:
  - 添加拼音格式示例
  - 增加地名 few-shot

  下一步:
  - 继续迭代 (说"继续")
  - 查看详情 (说"显示详细报告")
  - 停止 (说"停止迭代")
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 快捷命令

| 用户说 | 执行 |
|-------|------|
| "运行一轮评测" | 翻译 + 评测 |
| "开始迭代优化" | 完整迭代循环 |
| "继续迭代" | 继续下一轮 |
| "停止迭代" | 停止并保存状态 |
| "显示进度" | 查看当前状态 |
| "重置" | 清空状态，重新开始 |

## 文件结构

```
~/.openclaw/workspace/translation-eval/
├── data/
│   ├── gazetteer/         # 地名库
│   └── testset.json       # 测试集
├── prompts/
│   ├── baseline.md        # 基线 Prompt
│   └── iterations/        # 迭代版本
│       ├── v0.1.md
│       └── v0.2.md
├── reports/
│   ├── translations_*.json  # 翻译结果
│   └── eval_*.json          # 评测报告
├── state.json             # 当前状态
└── optimization_log.json  # 优化历史
```

## 注意事项

- 每轮迭代前保存当前状态
- 评测失败时回滚到上一版本
- 记录每次迭代的时间戳和结果
